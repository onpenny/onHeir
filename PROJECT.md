# OnHeritage - 遺產管理和安排平台

## 專案概述
一個全生命週期的遺產管理平台，解決資產信息孤島化、繼承主體不明確、生前資產遺忘漏記、繼承流程非標繁瑣等核心痛點。

---

## 一、當前遺產管理核心痛點

1. **資產信息孤島化**：銀行、保險、券商、基金、不動產、虛擬資產等分屬不同機構，無統一歸集入口，機構無主動告知繼承人的法定義務與通道，大量沉睡資產無法被繼承人查知。

2. **繼承主體不明確**：無標準化家族譜系存儲機制，旁系繼承人、非婚生子女、遺贈對象等信息易斷層，出現繼承權確權難、主體認定糾紛。

3. **生前資產遺忘漏記**：多類投資、另類資產（收藏品、股權、知識產權）零散分布，用戶生前自身都難完整盤點，身後更無跡可尋。

4. **繼承流程非標繁瑣**：不同地區 / 法域繼承法律、公證、過戶流程差異大，無一站式指引，繼承人時間、經濟成本極高。

5. **遺產捐獻無閉環**：捐贈意願登記、機構對接、憑證留存、進度追溯缺乏統一平台，意願落地難、透明度低。

6. **跨境繼承壁壘高**：海外銀行、保險、不動產等資產，受法律、語言、機構壁壘限制，繼承人幾無自主查詢路徑，資產流失率高。

7. **隱私與存儲矛盾**：傳統紙質 / 零散電子存儲易洩露、破損，兼具安全性與可繼承性的數字化解決方案稀缺。

## 二、核心基礎功能

### 2.1 全品類資產管理
- 對接金融機構 API，實現銀行、保險、理財等資產自動同步入賬
- 支援手動錄入非標資產：股權、收藏品、知識產權、虛擬貨幣、海外資產
- 生前定期資產盤點提醒，降低遺忘、漏記機率

### 2.2 智能家族譜系追蹤
- 可視化族譜搭建，兼容血緣、收養、婚姻等多元關係標註
- 預設法定 / 指定繼承人層級，關聯資產分配規則
- 分級權限管控，僅授權主體可查閱敏感信息

### 2.3 自動觸發與繼承通知
- 合規對接醫療、戶籍、社保數據，核驗生命狀態並觸發後續流程
- 按預設規則自動向繼承人推送資產清單、繼承指引
- 支援條件觸發通知（子女成年、完成學業等）、延遲通知

### 2.4 遺囑與分配規則管理
- 提供多地區 / 法域合規遺囑模板
- 可視化資產分配比例設置，支援動態條件分配
- 區塊鏈存證，強化電子數據法律效力

### 2.5 遺產捐獻全流程管控
- 財產、文物、遺體 / 器官等捐贈意願登記
- 對接正規基金會、醫療機構、文博機構，直連執行端
- 捐贈憑證、資金流向、項目進度實名追溯 / 匿名公示

### 2.6 一站式繼承工具
- 法域適配繼承材料清單、流程指引、文檔模板
- 在線公證、律師諮詢預約入口
- 跨境繼承法律條款檢索與基礎導航

### 2.7 隱私安全體系
- 端到端加密、操作留痕、多重身份驗證（生物識別、委託聯繫人核驗）
- 多節點備份，防止數據丟失與洩露

## 三、核心商業盈利點

### 3.1 訂閱制收費（穩定基礎現金流）
- **個人分層訂閱**：免費版、基礎年付版、進階年付版、高端尊享版
- **家族套裝訂閱**：捆綁多成員賬號，統一族譜與資產池管理

### 3.2 高毛利增值服務收費
- **法律與公證服務**：遺囑見證、繼承諮詢、糾紛代理、跨境法域法律意見書
- **遺產清算與代執行**：資產核驗、變現、過戶代辦
- **非標資產評估**：收藏品、藝術品、股權、知識產權評估對接
- **稅務籌劃服務**：遺產稅、贈與稅優化方案設計
- **沉睡資產尋回**：歷史資產專業查詢、確權、追回代辦（成功追回後按比例分成）

### 3.3 B 端合作授權與分成（規模化收入）
- **金融機構合作**：向銀行、保險、信託、券商輸出白標系統模塊
- **政企與機構項目制**：為民政部門、公證處、公益基金會定制的譜系 / 捐贈 / 繼承管理系統
- **跨境服務對接**：聯合海外律所、跨境財富機構，收取居間費 + 成功分成

### 3.4 衍生變現觸點
- **數字遺產與文創**：社交賬號、數字作品、影像託管與展示；族譜畫冊、家族傳承檔案等文創實物銷售
- **行業數據服務（合規匿名化）**：輸出遺產分布、繼承風險、捐贈意向等行業報告
- **精準產品導流**：生前規劃場景合規推薦終身壽險、家族信託等產品

---

## 四、差異化競爭亮點

1. **譜系 + 資產雙綁定**：區別於單一資產管理 / 族譜工具，實現關係與分配規則深度關聯
2. **主動觸發機制**：不同於傳統被動查詢平台，通過合規數據核驗實現主動通知
3. **全法域適配**：覆蓋境內多地區 + 主流海外國家規則，搶佔跨境繼承藍海
4. **捐贈全閉環**：從意願登記到執行追溯一體化
5. **生前死後全週期**：覆蓋規劃、預案、執行、傳承，非單一環節工具

---

## 五、合規與風險把控

- 嚴格遵循各地數據安全、個人信息保護法規（如 PIPL、GDPR）
- 明確電子遺囑、區塊鏈存證法律效力邊界
- 僅與持牌法律、金融、評估機構合作
- 建立完善的授權、註銷、數據刪除機制
- 流程全留痕，降低平台在繼承糾紛中的法律責任

---

## 六、冷啟動獲客思路

1. 聯合律師事務所、家族辦公室、保險經紀人，精準觸達高意向、高價值用戶
2. 推出「免費沉睡資產排查」活動
3. 針對中老年、高淨值群體做繼承風險、資產流失案例科普
4. 與殯葬、壽險場景捆綁，作為增值服務打包切入
5. 優先突破港澳台及跨境用戶
